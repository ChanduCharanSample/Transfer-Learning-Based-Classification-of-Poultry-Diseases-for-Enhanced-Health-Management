About the Project

This project focuses on detecting poultry diseases using deep learning and computer vision. Poultry farming is a critical component of the agricultural sector, but it is often affected by infectious diseases that can lead to economic loss and animal suffering.

🎯 Objective

To build an AI-powered system that can classify poultry images and predict common diseases using a trained deep learning model.

📊 Dataset

The dataset contains categorized poultry images labeled with conditions like:

  • Coccidiosis
  • Newcastle Disease
  • Salmonella
  • Healthy

🧠 Model Used

The model used for this project is VGG16, a pre-trained Convolutional Neural Network (CNN) which was fine-tuned for our dataset.

🔍 Features

  • Image upload and disease prediction
  • Confidence level with prediction
  • Detailed explanation for each disease

🌐 Deployment

The model is integrated into a Flask web application that allows users to interact via a web interface. Users can upload poultry images and instantly get disease predictions.

🚀 Outcome

This tool can be beneficial to farmers, veterinarians, and poultry businesses by enabling early detection and better disease management, thus reducing losses and improving productivity.
